# nodeJs-apk-consola
Aplicación de consola usando yargs.

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla de multiplicar.
                                               [número] [requerido] [defecto: 5]
  -l, --listar   Muestra la tabla por consola.
                                         [booleano] [requerido] [defecto: false]
  -h, --hasta    Define hasta donde va a ser la multiplicación.
                                                          [número] [defecto: 10]
```
